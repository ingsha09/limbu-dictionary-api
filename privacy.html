Privacy Policy

Last Updated: June 30, 2026

KisoLabs built the Limbu Dictionary app as a Free app. This SERVICE is provided by KisoLabs at no cost and is intended for use as is.

1. Information Collection and Use

Limbu Dictionary is designed to prioritize user privacy. The application operates primarily as an offline dictionary resource using public data. We do not collect, log, track, or share any personally identifiable information, location metrics, or search queries.

2. Network and API Requests

The application connects to the internet solely to fetch updated language dictionary structures from our open-access GitHub source dataset repository. No unique device data or user identifiers are attached to these dictionary fetch actions.

3. Local Storage and Caching

To ensure steady offline utility, the application caches parsed linguistic translations locally on your device via standard SharedPreferences. This data remains private and securely insulated within your application sandbox.

4. Changes to This Privacy Policy

We may update our Privacy Policy from time to time. Thus, you are advised to review this page periodically for any changes. We will notify you of any changes by posting the new Privacy Policy on this page.
